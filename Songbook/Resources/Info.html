What is Songbook?

Songbook is an organizer of Chord Pro files, a way of storing lyrics and chords of a song, for the iPhone or iPod Touch.

I wrote the application in my spare time in just three days.  Hopefully someone will find the App useful, and so I have decided to put it free in the Apple App Store.

Songbook is not a polished application yet, and it will certainly not win an ADA, but i think it is still better than some of the other applications in the App Store.

Being a student, and Songbook being a weekend project for me, I unfortunately cannot guarantee any regular update schedule.   This is the main reason Songbook is free.

How does it work?

The Songbook interface is straightforward with all navigation buttons at the bottom of the screen.

  • Artists: shows all the artists in your Songbook.
  • Songs:  shows all the songs in your Songbook.
  • Add Song: Either manually enter the text and chords for a song, or importing a Chord Pro file a URL
  • Info: shows documentation for Songbook

A Shortcut

If you are browsing internet with an iPhone or an iPod Touch and you find a chord pro file you want to add to your Songbook, simply substitute the http:// in the URL bar of Mobile Safari with chordpro://.  This will send the URL to Songbook, so you don't have to remember and rewrite it in the “Add Song” pane.

You can even add a bookmarklet that will do it for you.  Simple create a bookmarklet with the following javascript: 


javascript:window.location.href =
'chordpro' +
window.location.href.substr(4)

To create this bookmarklet, you must first save a bookmark from a webpage (for example from google).  Then delete the entire entry for the website URL and replace it with the javascript bookmarklet text above.  Change the name so you can remember what the bookmark is.  When navigating onto a page that is a Chord Pro file, simply click that bookmark and it will import the Chord Pro file into Songbook.

Editing a Song

In the Song Edit view (accessed by the top right button while viewing a song or ‘by typing it on the iPhone’ in the ‘Add Song’ menu) it is possible to edit a raw Chord Pro file. The format is really simple: first write or edit the lyrics of the song, then insert between the verses the chords surrounded by square brackets.

Since an example is worth a thousand words: Swing l[D]ow, sweet c[G]hario[D]t

Auto Scrolling Function

To access the Autoscroller function: open a song, click the menu button in the upper right-hand corner and click ‘Start Autoscroller’.  The speed is variable from 1-10
